Binary tree uint values on Solidity

1 add(uint value) - add new value to binary tree
2 getAmount() - get amount values in tree
3 getHead() - get tree head number (bytes32)
4 contains(uint value) - check contains value in tree
5 traversal(string typeTraversal) - traversing the tree by the following algorithms:
    5.1 - inOrderTraversal
    5.2 - preOrderTraversal
    5.3 - postOrderTraversal
6 getTraversal() - get an array and type of tree traversal
7 remove(uint value) - delete the value with the last reconstructed binary tree
